# CHaserOnlineNodejsServer
This is nodejs server for CHaserOnline2015.  

## Require  
・Nodejs  
・Couchdb which have these databases: chaser-server-sessions, users, rooms  

## How to use  
1.Install nodejs.<https://nodejs.org/>  
2.Install couchdb.<http://couchdb.apache.org/>  
3.Clone this repository. `git clone git@github.com:irc-archery/CHaserOnlineNodejsServer`  
3.Change current directory to this repository with Nodejs Prompt(windows) and use the command `npm install` in.
